CSS Effects and Animations

Used for enhancing the user interface.
CSS Transitions: Smooth changes in property values (transition: all 0.3s ease)
CSS Transformations: Rotate, scale, skew elements (transform: rotate(45deg))
Keyframe Animations: For custom animations (@keyframes)
CSS Filters: Apply visual effects (filter: blur(5px), grayscale(50%))
Box Shadow and Text Shadow: Create depth (box-shadow: 2px 2px 5px rgba(0,0,0,0.3))
