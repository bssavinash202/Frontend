CSS Layouts and Types

CSS offers multiple layout systems to structure web pages.
Box Model: Traditional layout approach
Flexbox: For one-dimensional layouts (display: flex)
Align items in rows or columns
Grid Layout: For two-dimensional layouts (display: grid)
Create complex grids with grid-template-rows and grid-template-columns
Positioning: static, relative, absolute, fixed, sticky
Float and Clear: Older method for layouts (float: left/right)
