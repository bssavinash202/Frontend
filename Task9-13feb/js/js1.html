Basics of JavaScript

Covers core concepts and syntax for beginners.
Variables: var, let, const
Data Types: Number, String, Boolean, Object, Array, Null, Undefined
Operators: Arithmetic (+, -), Comparison (==, ===, !=), Logical (&&, ||, !)
Control Flow: if-else, switch, for, while, do...while loops
Functions: Function declaration, expression, arrow functions (=>)
